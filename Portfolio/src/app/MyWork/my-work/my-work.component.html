Advisory System Rewrite

The hippo advisory system is a platform used to assist and advise customers on the best health insurance package available for them. The goal of this project was to rewrite the system that currently exists in order to make it more efficient, accessible and user friendly. My role in this project was to perform updates and queries on the database and to create endpoints that can be used by the frontend team to access the data they need from the database. I was also responsible for updating the queues associated with each API call and to occasionally store certain information in a cache. Technologies used for this project include .net core, SQL Server, RabbitMQ, Redis, Rancher, Magic Draw, Azure Functions, Kong, Docker, Xunit.
